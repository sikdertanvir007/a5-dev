Question-1: What are the different ways to select an element in the DOM?

Ans : To select an element I can use many different ways like getElementByID, getElementByClass, getElementByTagname and so on

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Ans : InnerHtml is used for to include HTML tags, innerText is used for get or set the visible text content of an element, textContent is used for to use raw text including text from hidden elements

Question-3: What is event delegation in the DOM?

Ans : Event delegation is a technique where I can attach a single event listener to a parent element, allowing it to handle events triggered on its child elements

Question-4: What is event bubbling in the DOM?

Ans : When an event triggered on an element propagates upwards through its parent elements in the DOM hierarchy until it reaches the root element
